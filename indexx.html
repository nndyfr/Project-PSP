Project

Aplikasi Pengelolaan Tugas

Aplikasi berbasis web yang dirancang untuk membantu pengguna dalam mengelola dan mengatur tugas harian. Diharapkan pengguna dapat menambahkan, mengedit, dan menghapus tugas serta menetapkan tenggat waktu. Proyek ini dibangun menggunakan HTML, CSS, dan JavaScript, dengan fitur database sederhana.

Analisis Data Penjualan

Proyek ini bertujuan untuk menganalisis tren penjualan toko online menggunakan data penjualan bulanan. Dengan visualisasi data menggunakan Python dan Matplotlib, proyek ini memberikan wawasan tentang produk terlaris dan waktu promosi terbaik.

Model Prediksi Harga Rumah

Membangun model machine learning untuk memprediksi harga rumah berdasarkan fitur seperti luas tanah, jumlah kamar, dan lokasi. Proyek ini menggunakan algoritma regresi linear dan dataset yang diambil dari Kaggle untuk melatih model prediksi yang akurat.
